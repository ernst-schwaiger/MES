# Setup

## Python venv
```shell
python -m venv venv
```

## Activate venv
```shell
.\venv\Scripts\Activate # on Windows
```

## Install packages for SUIT Tool
```shell
pip install cryptography
```

## Network
```shell
make network
```

## Generate Manifest
```shell
make all
```

## Generate Manifest in JSON-format
```shell
make json
```
